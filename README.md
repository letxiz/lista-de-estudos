# 📚 Lista de Estudos — Web App

Uma mini aplicação web desenvolvida com **HTML, CSS e JavaScript** para ajudar a organizar matérias ou assuntos de estudo.

A aplicação permite adicionar matérias à lista e removê-las quando desejar, facilitando o controle do que precisa estudar.

---

## 🚀 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## ✨ Funcionalidades

- Adicionar matérias ou assuntos para estudar  
- Remover matérias da lista  
- Impedir a adição de itens vazios  
- Interface simples e organizada  
- Playlist do Spotify para ouvir enquanto estuda  

---

## 🧠 Como funciona

1. O usuário digita uma matéria no campo de texto.  
2. Ao clicar no botão **Adicionar**, a matéria é inserida na lista.  
3. Cada matéria possui um botão **Remover**, que permite excluir o item da lista.  
4. O campo de texto é limpo automaticamente após adicionar a matéria.
